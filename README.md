# E-Commerce Website

A responsive e-commerce website built with React, featuring product browsing, cart management, routing, and Context API state management.

## Features
- Product listing
- Add products to cart
- Increase and decrease product quantity
- Remove products from cart
- Cart total calculation
- Responsive navigation bar
- Page navigation using React Router

## Technologies Used
- React
- JavaScript
- React Router
- Context API
- React Bootstrap
- Bootstrap Icons
- HTML and CSS

## Run the Project

```bash
npm install
npm start
